## 3. 这道题到底在考什么？

**哑节点**，用**哑节点（占位节点）**作为操作指针的起点，让代码能够顺畅地“挂”上新节点，最后通过 `return dummy.next` 去掉占位头节点，返回真正的结果链表。

两个相加的链表长度往往不一样。考的是程序员对**空指针（None）的边界防御能力**。必须知道在遍历取值时，如果链表走完变空了，要强行补 `0`，而不是直接去取 `None.val` 导致程序崩溃。**（本地代码里那个 `v1 = curr1.val if curr1 else 0` 就是核心关键点）**。

## 4. 手抄官方解法

class Node:
    def __init__(self, val):
        self.val = val
        self.next = None

def addTwoNumbers(head1, head2):
    carry = 0
    num = 0
    Newhead = Node(0)
    curr1 = head1
    curr2 = head2
    curr0 = Newhead

​    while curr1 is not None or curr2 is not None or carry != 0:

        # 【改1】先判断“空节点”取 0，防止程序崩溃

​        v1 = curr1.val if curr1 else 0
​        v2 = curr2.val if curr2 else 0
​        

        # 【改2】一次性算出总和，分离“当前位的数字”和“下一位的进位”

​        total = v1 + v2 + carry
​        num = total % 10     # 当前位存放的数字
​        carry = total // 10  # 给下一位的进位

        # 【改3】必须新建节点“接”上去，不能只改值（否则链表只有一节）

​        curr0.next = Node(num)
​        curr0 = curr0.next

        # 【改4】移动原链表指针时，要带上安全判断

​        if curr1 is not None: curr1 = curr1.next
​        if curr2 is not None: curr2 = curr2.next

​    return Newhead.next

def print_linked_list(head):
    if not head:
        print("None")
        return
    curr = head
    while curr:
        print(curr.val, end=" -> ")
        curr = curr.next
    print("None")

def make_linked_list(arr):
    if not arr: return None
    head = Node(arr[0])
    curr = head
    for val in arr[1:]:
        curr.next = Node(val)
        curr = curr.next
    return head

print("=== 测试 1：普通情况 ===")
l1 = make_linked_list([2, 4, 3])
l2 = make_linked_list([5, 6, 4])
result1 = addTwoNumbers(l1, l2)
print("输入: 342 + 465")
print("结果: ", end="")
print_linked_list(result1) 

# 预期输出: 7 -> 0 -> 8 -> None (807)

print("\n=== 测试 2：边缘情况（都是 0） ===")

l2 = make_linked_list([0])
result2 = addTwoNumbers(l1, l2)
print("输入: 0 + 0")
print("结果: ", end="")
print_linked_list(result2)

# 预期输出: 0 -> None

print("\n=== 测试 3：连续进位情况 (示例3) ===")
l1 = make_linked_list([9, 9, 9, 9, 9, 9, 9])
l2 = make_linked_list([9, 9, 9, 9])

print_linked_list(result3)
