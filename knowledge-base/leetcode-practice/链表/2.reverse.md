## 3. 这道题到底在考什么？
​    考的是**“三指针迭代法”**进行原地链表翻转。核心思想：
1. 需要三个变量：`prev`（前驱）、`curr`（当前）、`next_temp`（后继）。
2. 因为翻转 `curr.next` 时，会切断通往后续节点的线，所以**必须先拿 `next_temp` 把后面的节点存起来**。
3. 接着，把 `curr` 的箭头掉头指向前面的 `prev`。
4. 最后，三个指针整体往后平移一位，继续下一轮。
5. 循环结束时，`curr` 为 `None`，此时 `prev` 正好停留在**反转后的新头节点**上，因此返回 `prev`。

## 4. 手抄官方解法
我在本地跑通的完整代码：

```python
class Node:
    def __init__(self,val):
        self.val = val
        self.next = None
    
def reverse(head):


    if head is None:
        return head
   
    if head.next is None:
        return head
    
     
    prev = None
    curr = head
    next_temp = None
   

    while curr is not None:
       next_temp = curr.next
       curr.next = prev
       prev = curr
       curr = next_temp 

    return prev
    
node1 = Node(1)
node2 = Node(3)
node3 = Node(7)
node4 = Node(8)
node5 = Node(9)

node1.next = node2
node2.next = node3
node3.next = node4
node4.next = node5


def display(head):
    curr = head
    while curr is not None:
        print(curr.val,end = "->" )
        curr = curr.next

    print("None")

display(node1)

rev_node = reverse(node1)

display(rev_node)
```
