## 4. 手抄官方解法

我在本地跑通的完整代码：

```python
#定义节点
class Node:
# __init__   魔法方法 以这个为名的函数不需要手动调用，新建对象时自动调用，将值
#self 对象本身
    def __init__(self,value):
        self.value = value
        self.next = None

def append(head,new_val):
    new_node = Node(new_val)

    if head is None:
        return new_node

    curr = head
    while curr.next is not None:
        curr = curr.next

    curr.next = new_node

    return head  

##测试

node1 = Node(1)
node2 = Node(2)
node3 = Node(3)
node1.next = node2
node2.next = node3

# 现在调用你的 append 函数，把 4 接上
new_head = append(node1, 4)
new_head = append(node1,5)
# 打印验证一下结果
curr = new_head
while curr:
    print(curr.value, end=" -> ")
    curr = curr.next
print("None")
```
