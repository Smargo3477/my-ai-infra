# Linux 常用指令大全

## 一、文件导航与目录操作
| 命令 | 含义与常用用法 |
| :--- | :--- |
| `pwd` | 查看当前所在的绝对路径 |
| `ls` | 列出当前目录下的内容 |
| `ls -al` | 列出所有内容（含 `.` 开头的隐藏文件）及详细权限 |
| `cd 目录名` | 进入指定的文件夹（如 `cd projects`） |
| `cd ~` 或 `cd` | 直接跳回你的家目录（`/home/你的用户名`） |
| `cd ..` | 返回上一级目录 |
| `cd -` | 返回上一次所在的历史目录（来回切换超好用） |

## 二、文件与目录的增删改
| 命令 | 含义与常用用法 |
| :--- | :--- |
| `mkdir 目录名` | 新建一个文件夹 |
| `mkdir -p 路径/目录名` | 递归创建目录（即使父目录不存在也会自动创建） |
| `touch 文件名` | 创建一个空文件（若文件已存在，则更新其时间戳） |
| `cp 源文件 目标位置` | 复制文件 |
| `cp -r 源目录 目标位置` | 递归复制整个文件夹 |
| `mv 源文件 目标位置` | 移动文件（也可以用来重命名文件，即在同一目录下移动） |
| `rm 文件名` | 删除文件 |
| `rm -rf 目录名` | 强制递归删除整个文件夹（**极危险，慎用！** 尤其不要输错路径） |

## 三、查看与编辑文件内容
| 命令 | 含义与常用用法 |
| :--- | :--- |
| `cat 文件名` | 在终端直接打印显示文件全部内容 |
| `less 文件名` | 分页查看文件内容（按空格翻页，按 `Q` 键退出） |
| `head -n 10 文件名` | 只看文件的前 10 行 |
| `tail -n 10 文件名` | 只看文件的后 10 行 |
| `tail -f 文件名` | 实时动态滚动查看文件末行（看日志/错误报告极常用，按 `Ctrl+C` 退出） |
| `nano 文件名` | 在终端内直接打开简易文本编辑器编辑文件（编辑完按 `Ctrl+X`，按 `Y` 确认保存，再按回车退出） |

## 四、文本搜索与统计
| 命令 | 含义与常用用法 |
| :--- | :--- |
| `grep "关键词" 文件名` | 在单文件中搜索包含特定关键词的行 |
| `grep -r "关键词" .` | 在当前目录下，递归搜索所有文件包含该关键词的行 |
| `wc -l 文件名` | 统计该文件有多少行代码/数据 |

## 五、系统磁盘与资源查看
| 命令 | 含义与常用用法 |
| :--- | :--- |
| `df -h` | 查看磁盘分区情况（用 G/M 显示，直观看出硬盘是否满了） |
| `du -sh 目录名` | 查看指定文件夹占用了多少磁盘空间 |
| `free -h` | 查看当前内存使用情况 |
| `top` | 动态实时查看 CPU、内存和进程占用（按 `Q` 键退出） |

## 六、权限与进程管理
| 命令 | 含义与常用用法 |
| :--- | :--- |
| `chmod 755 文件名` | 修改文件权限（让文件可读可写可执行，常规用法） |
| `ps aux` | 列出当前系统运行的所有进程及其 PID |
| `kill -9 进程PID` | 强制结束指定的进程 ID |

## 七、网络与下载
| 命令 | 含义与常用用法 |
| :--- | :--- |
| `ping 网址` | 测试网络是否连通（按 `Ctrl+C` 停止） |
| `wget 网址` | 从网络下载文件到当前目录 |
| `curl 网址` | 发送网络请求，常用于获取接口数据或查看网页源码 |

## 八、压缩与包管理（Ubuntu）
| 命令 | 含义与常用用法 |
| :--- | :--- |
| `tar -czvf 压缩包名.tar.gz 文件夹/` | 将文件夹打包并压缩为 `.tar.gz` |
| `tar -xzvf 压缩包名.tar.gz` | 解压 `.tar.gz` 格式的压缩包 |
| `sudo apt update` | 刷新 Ubuntu 的软件源列表（**必须加 `sudo`**） |
| `sudo apt install 包名` | 安装指定软件包 |
| `sudo apt remove 包名` | 卸载指定软件包 |

## 九、历史与快捷操作
| 命令 | 含义与常用用法 |
| :--- | :--- |
| `history` | 查看你最近敲过的几百条命令历史 |
| `!数字` | 直接执行历史记录里对应数字行的那条命令 |
| `clear` 或 `Ctrl + L` | 快速清空当前终端屏幕显示 |
